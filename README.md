# data-quality-automation-portfolio
Upskilling projects in Python &amp; SQL for Data Quality and Automation

## 📋 Projects Overview

### Python Automation Scripts

1. **Log Parser Script**  
   Parses log files, identifies error patterns, and generates structured summary reports.

2. **Test Data Generator**  
   Creates realistic CSV test datasets with customizable edge cases using the Faker library.

3. **CSV Comparison Tool**  
   Compares two CSV files, highlights differences, and produces detailed reports (highly relevant to data migration and insurance data validation).

4. **API Test Script**  
   Performs API testing using Requests and PyTest for response validation.

### SQL Projects
- Advanced SQL practice on StrataScratch (medium-level) focusing on:
  - GROUP BY + HAVING
  - Window Functions (ROW_NUMBER, RANK, LAG, LEAD)
  - CTEs and complex data validation queries
- Building a reusable library of data quality SQL patterns (duplicate detection, NULL audits, gap analysis, etc.)
